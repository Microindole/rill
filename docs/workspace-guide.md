# Web 工作台使用

## 入口

- 页面：`/console`
- 需要登录后使用完整能力；游客可用受限能力

## 基础流程

1. 新建或选择会话
2. 输入 SQL，`Ctrl/Cmd + Enter` 执行
3. 查看结果与 trace
4. 将常用 SQL 保存为片段或场景

## 资产区

- SQL 片段：快速复用单条 SQL
- 场景脚本：保存多条 SQL 并按顺序运行
- 导出任务：将查询结果导出 CSV/JSON

## 导出任务

1. 在 SQL 区域准备语句
2. 点击“导出任务”
3. 选择格式（CSV/JSON）
4. 执行成功后可在任务列表点击“下载”
